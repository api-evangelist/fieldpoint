# Fieldpoint

Fieldpoint is an enterprise field service management (FSM) platform built for commercial contractors and service organizations across HVAC, electrical, fire inspection, security, medical equipment, and technology verticals. The platform covers the full service lifecycle including quoting, work order management, scheduling and dispatching, mobile technician apps, inventory tracking, service contracts, project costing, and customer management.

Fieldpoint exposes an open REST API known as the Fieldpoint Data Transfer Service (FPDTS) that enables custom integrations with ERP, CRM, accounting, payroll, and workforce management systems. The platform ships pre-built connectors for Microsoft Dynamics GP/365, NetSuite, QuickBooks, Salesforce, WorkMarket, AvaTax, and more.
